# 🧾 账单分类工具

一个用于自动读取、分类和整理支付宝、微信、银行交易流水的本地化桌面工具。

## ✨ 功能特点

- 📁 **多平台支持** - 支持支付宝、微信、银行账单格式
- 🔍 **自动识别** - 智能识别文件来源，无需手动选择
- 🏷️ **智能分类** - 基于19个预定义类别自动分类交易记录
- 🖥️ **图形界面** - 简洁易用的 GUI 界面，无需命令行
- 📊 **Excel导出** - 导出标准化的 Excel 报表（含分类汇总和明细）
- 🔒 **本地运行** - 数据不上传，保护隐私安全

## 🚀 快速开始

### 方式一：直接运行（推荐）

1. 确保已安装 [Python 3.10+](https://www.python.org/downloads/)
2. 双击 `install.bat` 安装依赖
3. 双击 `start.bat` 启动程序

### 方式二：命令行运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动程序
python src/main.py
```

## 📖 使用说明

1. **导入账单** - 点击「选择文件」或「选择文件夹」导入账单文件
2. **开始处理** - 点击「开始处理」进行解析和智能分类
3. **查看结果** - 使用分类筛选查看不同类别的消费
4. **导出报表** - 点击「导出 Excel」保存分类结果

## 📂 支持的账单格式

| 平台 | 格式 | 说明 |
|-----|------|------|
| 支付宝 | .csv | 支付宝导出的交易明细 |
| 微信 | .xlsx | 微信支付导出的账单流水 |
| 银行 | .xlsx/.csv | 各银行导出的交易记录 |

## 📁 项目结构

```
账单分类工具/
├── src/
│   ├── main.py              # 程序入口
│   ├── gui/                 # 用户界面模块
│   ├── parsers/             # 文件解析器（支付宝/微信/银行）
│   ├── classifier/          # 分类引擎（关键词匹配+规则引擎）
│   ├── exporter/            # Excel 导出模块
│   └── utils/               # 工具模块
├── config/                  # 配置文件
├── data/
│   └── keywords.json        # 分类关键词库
├── install.bat              # Windows 安装脚本
├── start.bat                # Windows 启动脚本
└── requirements.txt         # Python 依赖
```

## ⚙️ 自定义配置

### 修改分类关键词

编辑 `data/keywords.json` 文件，添加或修改分类关键词：

```json
{
  "餐饮": ["美团", "饿了么", "肯德基", "麦当劳", ...],
  "交通": ["滴滴", "打车", "公交", "地铁", ...],
  ...
}
```

### 添加分类规则

编辑 `config/categories.yaml` 文件，添加自定义分类规则。
