# DailyBill - 个人账单分类工具

一个用于自动读取、分类和整理支付宝、微信、银行交易流水的本地化工具。

## 功能特点

- 支持支付宝、微信、银行账单格式
- 自动识别文件来源
- 智能分类交易记录（19个预定义类别）
- 图形界面操作简单
- 导出标准化的 Excel 报表

## 安装

```bash
# 安装依赖
pip install -r requirements.txt
```

## 使用方法

### GUI 模式

```bash
python src/main.py
```

## 操作步骤

1. 点击"选择文件"或"选择文件夹"导入账单文件
2. 点击"开始处理"进行解析和分类
3. 使用分类筛选查看不同类别的消费
4. 点击"导出 Excel"保存结果

## 支持的分类

| 类别 | 说明 |
|-----|------|
| 旅行 | 机票、酒店、景点门票等 |
| 住房相关 | 房租、物业费等 |
| 餐饮 | 外卖、餐厅消费等 |
| 汽车 | 加油、停车、保养等 |
| 日用百货 | 超市购物、日用品等 |
| 备婚 | 婚庆相关支出 |
| 人情 | 红包、礼金等 |
| 交通 | 打车、公交、地铁等 |
| 数码电器 | 手机、电脑、家电等 |
| 文化休闲 | 电影、游戏、书籍等 |
| 服饰 | 服装、鞋帽等 |
| 通讯 | 话费、宽带等 |
| 家居家装 | 家具、装修等 |
| 水电燃气费 | 水费、电费、燃气费 |
| 医疗 | 药品、挂号费等 |
| 美妆护肤 | 化妆品、护肤品等 |
| 运动户外 | 健身、运动装备等 |
| 学习 | 课程、培训等 |
| 其他支出 | 无法归类的支出 |

## 项目结构

```
DailyBill_2/
├── src/
│   ├── main.py              # 程序入口
│   ├── gui/                 # 用户界面
│   ├── parsers/             # 文件解析器
│   ├── classifier/          # 分类引擎
│   ├── exporter/            # 导出模块
│   └── utils/               # 工具模块
├── config/                  # 配置文件
├── data/                    # 数据文件
├── tests/                   # 测试
└── requirements.txt         # 依赖清单
```

## 自定义分类

可以通过修改 `data/keywords.json` 文件来自定义分类关键词。
